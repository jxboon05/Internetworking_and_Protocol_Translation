# Advanced Internetworking: DHCP Relay & NAT-PT Protocol Translation

A networking project demonstrating enterprise DHCP Relay deployment, Port Address Translation (PAT), and IPv4-to-IPv6 protocol translation (NAT-PT/NAT64). The project was implemented using Cisco Packet Tracer and GNS3 to simulate real-world enterprise networking environments.

---

## Technologies Used

- Cisco Packet Tracer
- GNS3
- Cisco IOS
- Wireshark
- QEMU Microcore Linux

---

## Key Features

### Cisco Packet Tracer

- Configured DHCP Relay (`ip helper-address`) across different subnets
- Implemented PAT (NAT Overload) for Internet access
- Verified DHCP address assignment using the DORA process
- Tested end-to-end connectivity between internal networks

### GNS3

- Configured static NAT-PT for IPv4 and IPv6 communication
- Implemented dual-stack routing
- Configured QEMU Linux hosts using Linux networking commands
- Verified protocol translation using ICMP and ICMPv6
- Captured and analyzed packets using Wireshark
